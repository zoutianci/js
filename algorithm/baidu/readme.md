###二分查找
二分查找的时间复杂度: O(log2n)

1-1000 正整数  要猜多少次? 二分法

二分查找是一种分治思想的算法
1. 数组中排在中间的数字A 与它进行比较大小
2. 因为数组有序,
  a. A较大, 说明应该从前半部分查找
  b. A较小, 从后半部分查找
3. 递归 前两步

WeakMap的作用:
1. Dom 对象的操作   
2. 做缓存

js 要做到计算缓存 -> 闭包 -> 数组或对象存储
